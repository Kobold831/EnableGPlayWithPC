## 注意

> [!NOTE]
> チャレンジパッド2シリーズでは、GAppsを正常に動作させるためにDchaServiceを使用します。（一部の機能は使用できません）
> 
> ※デバッグ中のため将来のバージョンで提供予定
> 
> チャレンジパッド3､ Neoでは、GAppsを完全に動作させることはできません。  
> チャレンジパッドNextの場合は、**完全に動作できます**。
